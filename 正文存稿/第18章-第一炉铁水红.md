# 第18章 第一炉铁水红

废石山回来的第三天，阿陶没睡过一个整觉。

他在炉前蹲了又蹲，炭换了三茬——松木炭火软，橡木炭火硬，可都化不开那块铁矿石。铁石蹲在火里，烧红了皮，敲开来芯子还是黑的。阿陶眼下两团乌青，攥着石锤，嘴唇咬得发白。芽芽端了碗粟粥过来，他摆摆手，眼没离开炉膛。

"头人，铜的化点低，柴火就够。铁不行——它要的是一股狠火。"阿陶拿树枝在地上划：炉膛、风口、炭层。他画了擦，擦了画，像第一次画轮子时那样，脑子里的东西急着往外蹦。

我蹲下看他画。文明6里，炼铁术有个前置科技——鼓风炉。游戏里点一下就解锁；这儿，得把风吹进炉膛，把炭火逼到铁肯低头的热度。我忽然想起废石山坑道里那黑炭渣——旧纪元的人，拿什么配的炭？

"试试那炭渣。"我说。

阿陶把炭渣碾碎，混进橡木炭里，点火，拉起羊皮风囊——那是上回做大壮鞍具剩的皮子，缝成囊，两头开口，一拉一推，风呼呼往炉膛里灌。火苗窜起来，先是橘红，再是亮黄，末了泛白。阿陶死死盯着，汗珠子顺着下巴滴进火里，嗞一声化成气。炉膛里那声音变了——噼啪声没了，换成了闷闷的嗡。

"到了！"阿陶喊。

铁矿石在火里变了色。先是暗红，再是亮红，末了像太阳碎了一块掉进炉膛——白亮得不敢直视。阿陶拿火钳夹出来，搁在石砧上，抡锤。第一锤下去，铁石扁了；第二锤，火星溅了一屋子；第三锤——渣子碎了，底下露出一块青灰色的铁。

华夏第一块铁。

阿陶捧在手里，烫得直吹气，却不撒手。围观的族人先是静，后是炸——老石头独眼瞪得溜圆，老阿木把铜斧往地上一顿，蹲下就哭。这汉子砍过半座林，从没掉过泪。他呜咽着说："头人，咱以后不用怕铜甲大部了。"

那块铁从阿陶手里传了一圈。每个人接过去都掂一掂，摸一摸，像摸一个刚出生的娃。铁的凉，铁的沉，铁的棱角——华夏人第一次用自己的手摸到这种东西。石头不对，铜也不对——是铁。芽芽也伸手摸了摸，缩回来，瞪大眼："头人，它凉得跟溪水似的，可灰灰不敢闻。"灰灰在脚边打了个喷嚏，尾巴夹着，一脸不服。

我望着那块铁，心里那本账翻得哗哗响。从铜到铁，文明跃了一整级。铜是捡来的——矿石化了浇模，老天爷给的；铁是逼出来的——得鼓风、得配炭、得把火逼到连石头都扛不住的热度。一个文明能不能炼铁，全看炉前的人肯不肯蹲三宿不睡。

面板忽然跳了——

【铁器时代前置科技已解锁：铁制兵刃 / 铁制农具 / 铁制工具】

我怔了一下。冶金尤里卡，跟制陶那次一样——面板不施舍，只照见。照见的是阿陶三宿没合的眼，和他手里那块还烫手的铁。

阿陶没停。他趁着炉火正旺，把剩下的铁矿石一块块往里送。风囊呼呼响，锤声当当响，从午后敲到天黑。几个年轻汉子凑过来给他拉风囊、搬矿石——阿陶收了三个徒弟，都是石部落来投的后生。他手把手教：炭要碾多细，风要拉多快，铁要烧到什么色才算火候。徒弟们学得认真，一个拉风囊的拉得太猛，火苗窜出来燎了眉毛，众人哄笑。阿陶也笑了——他上一次笑，是烧出第一只盆的时候。第二块铁打成刀——青灰刃口，比铜刀轻，往铜刀上一磕，铜刀豁了口。第三块打成锄——比铜锄薄，入土却利落，老石头接过去在墙角试了试，一锄下去，硬土裂成两半。他回头看我，独眼里有水光："头人，这锄头，能多开出两片地。"

铁器从炉膛里往外蹦——刀、锄、斧、凿。每一件都在手里掂出不一样的分量：铜器是钝的，铁器是利的。铜器砍三下才倒的树，铁斧一下。铜锄刨半天的地，铁锄一炷香。大壮把铁刀别在腰上，拔出挥了两下，风声都变了——铜刀挥起来是"呼"，铁刀是"嗖"。他又把铁刀递回给阿陶，认真道："给我打把长些的，马背上抡的。"阿陶点头，在石板上刻下尺寸。华夏第一张铁器订单，是大壮的骑兵刀。

芽芽给铁锄起了个名："叫它'快锄'，比石头爷爷那把利多了。"老石头拿"快锄"在墙角试了试，一锄下去硬土裂成两半。他回头看我，独眼里有水光："头人，这锄头，能多开出两片地。"

夜里，炉火还没熄。铁器一字排开在炉前——铁刀三把、铁锄两把、铁斧一柄、铁凿两根。火光映在青灰刃口上，每条刃都泛着冷光。华夏第一炉铁，拢共八件。不多，可每一件都够换十条铜的命。

阿陶靠着炉壁睡着了，脸上黑一道白一道，手里还攥着块铁矿石。芽芽轻手轻脚给他盖上张皮子，灰灰蜷在他脚边，尾巴搭在他脚背上。灰灰忽然起身，叼了块碎铁片搁在芽芽手心，歪头看她。芽芽笑："灰灰说，这块归我。"她把铁片揣进怀里，拍两下——那是放薯干的地方，如今多了块铁。

三个徒弟轮流守炉，往炉膛里添炭，风囊时不时拉一下，让火吊着不灭。第一炉铁炼出来了，往后每一炉都是往上堆的砖。

我站在炉前，望着炉膛里暗红的余烬，忽然冒出一句——铁是土里长出来的，也是人熬出来的。

老姜从墙根踱过来，独眼映着炉火的残光。他捡起块边角料——打刀剩下的碎铁片，在指间翻来覆去，半晌："头人，咱华夏，从石头里拽出了铁。这往后，世道得听咱的。"

我没接话。铁握在手里，是热的，沉甸甸的像握住了一条命。可铁也招风——炎部落的炉烟还挂在南边，铜甲大部还在并吞小族。握住了铁，只是第一步。下一步，是让铁说话。让铁刀告诉南边：这儿有座城，城里有炉，炉里有火，火里有铁。

天快亮了。炉膛里最后一点余烬灭了，阿陶还在睡，芽芽靠在他肩上也睡着了，灰灰竖着一只耳朵，像替他们守夜。阳城的第一炉铁，就这么烧进了夜。明早太阳出来，铁器就会挂上每个人的腰——华夏，翻过了一页。
