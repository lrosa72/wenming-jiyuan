# 第81章 归途上的异乡人

> *「我们最终都会成为别人的回忆。」——安德烈·塔可夫斯基*

第二十六天，我们撞上了坛城车队的尾巴。

当时我们在一条干涸的河谷里找水——巴图说这条河的源头在雪线附近，夏天化雪的时候有水，秋天就干了。我们沿着河谷走了半个时辰，拐过一个弯的时候，迎面撞上了一个人。那人身上裹着一张发黑的旧羊皮，赤着脚，脚底的茧厚得和鞋底一样。他蹲在河床边用手扒石头找水，听到马蹄声抬起头来看了我一眼，然后愣住了。

坛城的人。一张坛城人的脸——颧骨高、眼窝深、额头上有一个烧灼过的烙印，是弯钩符号——他们从小就烙上的。

他站起来，往后踉跄了一步，然后做了我完全没有预料到的事：他没有跑。他跪了下来。无关恐惧——那是一个走了一辈子路的人终于看到人的那种跪。他跪在干涸的河床上，嘴张开又合上，最后说了一句我勉强能听懂的话：

「后面的路上……还有很多人。」

那一天我们遇到了三十一个坛城难民。有老人，有背孩子的妇人，有握着断铁棍的年轻人。他们没有武器，没有方向，只是在走——往西北走，朝着零号遗迹的方向。他们说坛城的首领在七天前接到了一封信，信上只写了两个字：「走吧。」首领看完信之后下令：全城北迁。不留人。东西能带多少带多少，带不走的丢了。

三天之后我们遇到了第一批尸体。一个老人靠着路边的一块石头坐着，头低垂，双手交叠放在膝盖上——像是睡着了。但胸口没有起伏。他身上盖着一件干净的外衣，是有人临走前替他盖上的。他们没有时间埋他。他们只能给死人盖一件干净的衣服，然后继续走。

大壮蹲下来检查了老人身边的地面——没有挣扎痕迹。只是走不动了，坐着休息，然后没有醒。老人的掌心里握着一块陶片——陶片上刻着一个弯钩标记。坛城的符号。他把自己的身份证明握在手里，握着它走完了最后一程。

阿雅把她的水囊解下来，把剩下的一半水倒在老人面前的土里。她没说话。
