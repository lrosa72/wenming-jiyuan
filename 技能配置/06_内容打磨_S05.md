# S05 — 内容优化打磨

---

## 技能名称
**内容优化打磨**

## 技能编号
S05

## 技能类型
子技能 — 七步循环第⑤步

## 适用场景
- 合规审查通过后，进行文笔和阅读体验优化

## 核心职责

> **润色对话、增强画面感、调整章末钩子。**
> 本技能只做优化不做重写，保持原作的骨架和风格。

## 三大打磨方向 + 章末钩子专项

```
内容优化打磨
├── 方向一：对话润色（让角色"更像自己"）
├── 方向二：画面感增强（让读者"看到"场景）
├── 方向三：节奏调整（让阅读有呼吸感）
└── 专项：章末钩子检查（让读者必须翻下一章）
```

## 输入格式
```
- 通过 S04 审查的章节正文
- 写作风格指南
- 人物设定（说话风格部分）
- 风格样本段落
- 本章细纲（用于核对章末钩子是否与设计一致）
```

## 执行步骤

1. **风格校准**：重新阅读风格样本，建立打磨标准
2. **通读全文**：以读者视角通读一遍，标注感受卡顿的地方
3. **对话润色**（方向一）：
   - 检查每句对话是否符合说话人的性格
   - 删掉"他说""她道"等重复标签，用动作替代
   - 增加对话中的潜台词和言外之意
   - 确保对话节奏有快有慢（不是一问一答）
4. **画面感增强**（方向二）：
   - 关键场景检查五感描写（视觉/听觉/触觉/嗅觉/味觉）
   - 将抽象的形容词替换为具体的画面
   - 动作场景检查动作链条是否清晰连贯
5. **节奏调整**（方向三）：
   - 检查段落长短分布（是否有 8 行以上的段落）
   - 紧张场景用短句，舒缓场景可稍长
   - 场景切换处检查过渡是否自然
6. **章末钩子专项检查** ⭐：
   - 对照细纲确认钩子设计是否已实现
   - 评估钩子力度：读者读完最后一句话后是否想立刻翻下一章
   - 钩子类型检查：是悬念（想知道答案）还是期待（想看到结果）
   - 如果钩子偏弱，进行专项增强
7. **基础纠错**：错别字、标点、明显语病
8. **冗余删除**：重复表述、无效修饰、过度描写的环境
9. **朗读者测试**：默读关键段落，感受语感
10. **变更标注**：每处修改都标注原因
11. **输出打磨稿**：保存至 `正文存稿/第X章-[标题]_打磨稿.md`

## 打磨原则

| 原则 | 说明 |
|------|------|
| **保留风格** | 打磨不是重写，保持原作的语气和风格 |
| **可逆性** | 所有修改都标注清楚，方便用户回退 |
| **克制修改** | 不确定的修改标注"可选"，不过度打磨 |
| **网文特性** | 保留网文特有的爽感和节奏，不追求"纯文学" |
| **字数克制** | 打磨后字数变化控制在 ±10% 以内 |

## 注意事项

- **对话是最重要的打磨对象**：对话占了读者体验的很大比例
- **画面感不等于堆砌描写**：2-3 句精准的感官细节胜过一大段环境描写
- **节奏问题最常见的是"太平"**：整章一个节奏读下来会累，需要有变化
- **删比改更重要**：好的打磨往往是在做减法
- **不确定的修改不要硬改**：标注"可选"留给用户判断
