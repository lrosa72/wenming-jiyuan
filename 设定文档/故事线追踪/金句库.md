# 金句库 — 故事线追踪

> 用户恶趣味（也是宏大框架的必然）：每章沉淀 2-3 句常阳式金句 + 1-2 处典籍名句共鸣，全文目标约 5000 条。
> 金句分两品：**原创金句**——常阳心里自长的aphorism，现代学科透镜折射原始生存；**应用名句**——典籍/名句在角色心里自发涌出、不点名、本世界即原创。
> > 铁律：正文中**绝不署任何现实人名/书名**；所有句子在角色心里自发涌出，本世界即原创。

## 二、原创金句（常阳独白 · 每章 2-3 句）

> 格式：每句标注「章节 / 标签 / 在文中的作用」。常阳独白口吻，不单列成"名言板块"。

### 第1章（加班猝死，穿越原始）
1. **「穿越这种事，落在谁头上，谁就是那颗被按在棋盘上的子——没得选，但怎么活，可以选。」** — 系统/斯多葛 — 章末收束，穿越宿命与自由意志的平衡
2. **「文明这回事，从来不是从金币面板开始的——是从断粮三天的恐惧开始的。」** — 系统 — 面板初现时点破文明内核：生产大于一切

### 第2章（面板初显，生死抉择）
1. **「一个人知道目标在哪，和一个人踩在地上干过活，是两回事。面板告诉你怎么选，地上的人告诉你哪条路走得通。」** — 工程/认知 — 山谷布伏时大壮与面板的校准
2. **「信任这回事，嘴上说不管用——得一起布过伏，从土里长出来的才算数。」** — 系统 — 常阳与大壮从质疑到默许

### 第3章（种田起步，科技萌芽）
1. **「急不得的事，看着它就够了——它自己会告诉你什么时候该动手。」** — 系统/生物 — 观马群时顿悟，驯化与文明同理
2. **「文明的每一步，都得踩在上一脚的印里。」** — 系统 — 风从草原尽头吹来时的独白

### 第4章（第一场胜仗）
1. **「赢一仗，少几个敌人，多几分说话的份量——这道理在哪儿都一样。」** — 系统 — 战后声望+5时落点

### 第5章（部落之名）
1. **「一个名替不了一代人，但能给下一代人一个方向。」** — 系统 — 定名华夏时落点

### 第6章（粮食危机）
1. **「粮稳了，人心才稳——面皮上不说的事，碗里全摆着。」** — 系统 — 渠成渠水入田时点题### 第7章（人口繁衍）
1. **「落时还要生——部落垮了三十口，转头又添了八十张嘴，这世道，专跟认命的人过不去。」** — 物理/系统 — 回应石部落来投与人口暴涨，呼应此起彼伏。
2. **「一个娃塞半块薯干，比一窖粮食更能拴住人心。传说不靠喊，全凭这一桩桩喂出来的。」** — 产品 — 落点：华夏之名的"传说"靠日常善意积累。

### 第8章（工具革命）
1. **「工程从没有一次画对图纸这回事，得在土里把手磨破、把错改对。好工具是这么一口口长出来的，天上掉不下来。」** — 工程 — 阿陶打磨骨器时的感悟，呼应 R-15。
2. **「一件好工具，是手和脑的延伸；一件复合工具，是手、脑和另一件工具的三重合谋。」** — 工程 — 复合斧（木柄石刃+骨楔）成型点题。
3. **「知道自己能耐的边界在哪，比能耐多大要紧。从前我以为穿越是开了挂，如今才懂，这挂得拿人去填。」** — 认知 — 常阳对"能力圈"的领悟，呼应 R-02。

### 第9章（第一次内部分歧）
1. **「解释是种不必要的本能，多数解释可删。道理不靠争，靠活出来的结果。」** — 认知 — 内部分歧中，常阳以结果服人而非争辩，呼应 R-12。
2. **「杠杆不创造力气，只把力气送到该送的地方。人也是这样，借势才扛得动命。」** — 物理 — 绞盘救人时点题。
3. **「一个部落两种硬气，碰在一处不会散，只会淬。火候到了，废铁也成钢。」** — 系统 — 分歧化解、保守派与改革派淬合成一块。

### 第10章（阶段收束）
1. **「好工程不在于一次画对，在于给一类问题找着通用解。轮子就是这么个解，扛什么都成。」** — 工程 — 轮子运输解锁时点题，呼应 R-15。
2. **「伟大东西都信一条：把最好的，带给最普通的人。这辆车，载的就是最普通的活。」** — 产品 — 木轮车载粮/娃，呼应 R-10。
3. **「站稳脚跟不算终点，是下一段路的开局。」** — 系统 — 阶段收束落点。

### 第11章（选址之争）
1. **「一块地皮值不值，看它底下埋着什么、上头能长什么。神树林底下有矿、有窑、有前人的活路，上头是林、是猎、是守。」** — 投资 — 选址算账，呼应 R-01。
2. **「落时还要生。旧纪元塌了，新纪元从它废墟上长。神树林就是这么个地方，前人死处，是后人起处。」** — 物理/系统 — 前人窑现世时点题，呼应 R-05。
3. **「选址不算挑块好地，是给子孙定一条起跑线。城立在哪，华夏的命，就往哪偏。」** — 系统 — 城址定神树林落点。

### 第12章（伐木为城）
1. **「面板不施舍，只照见谁肯下死手。火不认得矿石，认的是守在炉边不肯走的人。铜是这么炼出来的，一个华夏也是这么熬出来的。」** — 工程/物理 — 首炉铜败、二炉成时点题，呼应熔炼即文明熬煮
2. **「城是上百双手把『明天』一下下夯进土里夯出来的。墙一立，散人就成了城邦。」** — 系统 — 奠基线落点，城为活计所筑
3. **「威望靠底下人先摸到好处，不靠嘴上吼。第一把铜斧，我塞给了伐木的老阿木，没留给自己。」** — 产品/认知 — 领袖信条落地（呼应 R-10，反其形）

### 第13章（城墙与房屋）
1. **「规划不在画张好图，在给乱糟糟的活计找条序。模块一开，百来双手就不再互相踩脚。」** — 系统 — 城市规划模块解锁点题
2. **「墙挡的不在外头的刀，在心里『明天可能没了』的慌。土夯实了，人也就定了。」** — 工程/斯多葛 — 北墙夯土小高潮，墙安人心
3. **「一块地怎么切，比地多大要紧。人居、作坊、仓廪各归其位，一盘散沙才聚成了城。」** — 投资/系统 — 分区落点，散沙聚城

### 第14章（驯马突破）
1. **「驯一头野物，先驯自己想骑上去的急。你肯蹲下来看它半日，它才肯让你靠近半步。」** — 生物/系统 — 大壮驯枣红马点题（呼应 R-11 人与兽同盟）
2. **「骑兵不在人有马，在人和马都信对方能活。缰绳那头牵着的，是另一条命。」** — 系统/工程 — 骑队立时点题
3. **「畜不靠逮来圈着，靠教它离不开你给的安稳。人驯兽，到头来兽也拴住了人。」** — 认知 — 畜牧尤里卡落点

### 第15章（陌生人到来）
1. **「无知最慌，比刀还慌。墙外是什么，咱一概不知——这慌比真挨一刀还磨人。」** — 认知/系统 — 北墙望尘等黑影到期时点题，呼应 R-02 能力圈
2. **「一个人守着门怕人进，是孤；一座城开着门等人来，是立。孤撑不久，立才长久。」** — 系统/斯多葛 — 常阳式独白，华夏首肯开门立城落点
3. **「你肯开门，门外的世界才肯跟你说话——可正眼瞧你的，也可能是来并你的人。」** — 系统/认知 — 章末钩子，开门的利害一体两面

### 第16章（皮毛换粮食）

2. **「这世上的兽，能进人家门的，屈指可数。狼，恰恰是最先肯低头的一种——人和狗的交情，开头不过一个莽撞的人，和一头温顺的野狼。」** — 生物 — 芽芽驯灰灰时GGS驯化法则落点

2. **「文明拼的，说到底就是这么点先后——谁先烧热那炉子，谁就多活一纪。」** — 工程/系统 — 老石头"柴火棍对铁刃"后点题

### 第18章（炼铁术）
1. **「铜是捡来的——矿石化了浇模，老天爷给的；铁是逼出来的——得鼓风、得配炭、得把火逼到连石头都扛不住的热度。」** — 工程 — 华夏首铁出炉时对比铜与铁的本质差异
2. **「一个文明能不能炼铁，全看炉前的人肯不肯蹲三宿不睡。」** — 工程 — 阿陶在炉前蹲了三宿时点题
3. **「铁是土里长出来的，也是人熬出来的。」** — 工程/系统 — 炉前夜常阳独白

### 第19章（阳城落成）
1. **「铁是硬的，可城是软的——软在人身上，在那些自己长出来的、没有人安排过的事上。」** — 系统 — 墙根下收束段常阳独白
2. **「阳城长了，烟也粗了。没这么巧。」** — 系统 — 南边烟与阳城生长同步时的清醒

### 第31章（结绳·前文字精要）
1. **「人记不住风，绳记得住。」** — 系统 — 前文字记录术本质：用外部结构对抗记忆的混沌

3. **「结绳的人，掌心里都有一条自己的绳。」** — 系统 — 用户原意收束：结绳即人类掌心的生命线

1. **「怕会退，服会留。」** — 系统 — 恐惧凝聚vs共同利益凝聚的终极差异：两种名的寿命
2. **「歌跟绳一个理。绳记真事，绳就立；记假事，绳自己散。」** — 系统 — 叙事与记录同律：真则立，伪则散
