# 写作风格指南 — 《天地为棋，文明纪元》

> 本文件为 S03（正文生成）和 S05（内容打磨）提供风格基准。

---

## 一、叙事视角

**第一人称** — 常阳视角，严格使用「我」。不允许随意切换到第三人称。

- 读者通过常阳的眼睛看世界
- 常阳不知道的信息，读者也不知道（除非通过面板提示）
- 内心独白是重要组成部分（游戏玩家的吐槽、分析、算计）

---

## 二、语言风格

**男频主调：热血粗粝 + 稍带口语化 + 现代游戏玩家腔**

> 基调转向男频——节奏狠、感官糙、带点嘴上不干净的真性情。猫腻式的「凝练白描」保留为**写作技法**（见六·六），但不再是整体语气主锚。

| 特征 | 说明 |
|:---|:---|
| 节奏 | 快、狠、短。段落 2-4 行，战斗章可 1-2 行一句，手机阅读如刀 |
| 语气 | 口语化、带劲、稍带粗口（他娘的/老子/这帮），但「稍带」不泛滥 |
| 热血 | 推进靠冲劲与胜负欲，主角算计里裹着狠劲，不温吞 |
| 粗粝 | 多写血汗土铁腥，少写文艺比喻；用硬物象，不铺柔光 |
| 幽默 | 现代社畜吐槽保留，但收着，严肃章直接掐 |

---

## 三、对话风格

| 角色 | 说话风格 |
|:---|:---|
| 常阳 | 现代口语，带游戏术语（「这波尤里卡太关键」「要卡科技树了」），思考时自言自语 |
| 老族长 | 古朴稳重，多用比喻和部落谚语 |
| 大壮 | 简短有力，很少废话，行动派 |
| 阿陶 |  excited，经常「哦！我懂了！」 |
| 巴图 | 圆滑世故，商人腔调，爱说「在我的旅途中……」 |

**对话占比**：30-40%

## 四、描写密度

| 类型 | 比例 | 说明 |
|:---|:---|:---|
| 环境描写 | 15% | 2-3句精准感官细节，服务于氛围 |
| 心理描写 | 25% | 主角的分析、算计、吐槽 |
| 动作描写 | 30% | 战斗、建设、操作面板 |
| 对话 | 30% | 推进剧情和展示性格 |

## 五、节奏偏好

- **快节奏开场**：前300字必须进入状态（冲突/悬念/动作）
- **章末留钩子**：不让读者有「可以停下来」的感觉
- **爽点密度**：每章至少1个小爽点（面板数据暴涨/科技突破/敌人吃瘪）
- **长短句交替**：刻意制造长短句变化，避免AI味的对称结构

## 六·五、句式硬规则 ⭐⭐⭐（最高优先级）

> **彻底禁用「不是……，是……」句式。**

沿用前序「炉石小说」项目的明确规则，无任何例外。

### 禁止形式

| 形式 | 示例 | 违规 |
|:---|:---|:---|
| 标准式 | 「不是椅子，是泥地」 | ❌ |
| 并列式 | 「不是空调味，也不是酸臭味」 | ❌ |
| 拆分式 | 「不是普通的闪烁。是一道光斑。」 | ❌ |
| 因果式 | 「不是因为胆子大，是因为……」 | ❌ |

### 修正方法

用**白描 / 直接陈述 / 意象对比**替代，绝不出现「不是」与「是」的对应结构。

| 违规 | 修正 |
|:---|:---|
| 不是椅子，是泥地 | 身下是铺着兽皮的硬泥地 |
| 不是普通的闪烁。是一道光斑 | 屏幕炸开一团白光 |
| 不是因为胆子大，是因为…… | 胆子不大。脑子里那根弦却转得飞快 |

### 自检清单（每章必查）

- [ ] 全文搜索「不是」，确认无「不是X，是Y」结构
- [ ] 全文搜索「不是」+「是」，确认无跨句拆分变体
- [ ] 全文搜索「也不是」「也不是Y」，确认无并列变体

## 六·六、男频文风主锚 ⭐⭐⭐（热血粗粝 · 口语化）

整体语气走男频：冲、糙、带点嘴上不干净的真性情。猫腻的「凝练/白描/短句」降为**手艺**，融进男频骨架里，不取其温吞文艺的整调。

| 维度 | 男频表现 | 本章落点 |
|:---|:---|:---|
| 热血冲劲 | 推进靠胜负欲与狠劲，主角算计里裹着冲 | 战斗章句短如刀，胜了要爽到 |
| 粗粝感官 | 多写血/汗/土/铁锈/骨，硬物象，不铺柔光 | 落石、火、血溅用硬词 |
| 稍带口语 | 他娘的/老子/这帮/哥几个，克制不泛滥 | 常阳内心与短吼用口语 |
| 白描（手艺） | 用动作/物象说话，不直述情绪 | 「恐惧」用族人低头写 |
| 短句呼吸 | 三两行一断，战斗可一句一行 | 段落长短有杀气 |
| 意象克制 | 以具象代抽象，但意象要硬 | 面板红光→「血一样的红漫上来」 |
| 不滥情 | 悲壮暗写，不点破，不矫情 | 前一任首领之谜用老族长眼神 |
| 收幽默 | 现代吐槽保留但收，严肃章直接掐 | 战争/牺牲章不抖机灵 |

### 普通写法 → 男频式（热血粗粝 + 白描）

| 普通写法 | 男频式 |
|:---|:---|
| 他感到很恐惧 | 他低了头，手指抠进泥里，指节发白 |
| 这让他非常震惊 | 他愣了片刻，瞳孔缩成针尖 |
| 常阳心里想：这太荒谬了 | 常阳盯着腕上的光，半天没蹦出一个字 |
| 环境很荒凉 | 风卷黄土，山脉的尽头看不见 |
| 他赢了，很高兴 | 他抹了把脸上的血，咧嘴，牙全是红的 |
| 蛮族很可怕 | 那帮蛮子眼珠子发绿，像饿了一冬的狼 |

## 六·七、人味与故事性 ⭐⭐（用户反馈增补）

> 用户指出：文章流畅但缺点人味、故事性不强、讲述技巧平淡。修订方向如下。

| 维度 | 要求 | 落点 |
|:---|:---|:---|
| 人味 | 角色像活人，有软处、有温度 | 引入常驻暖色角色（芽芽·六岁孤女）；常阳冷硬外壳下的软；族人小互动（老石头藏肉干、阿陶捏泥马） |
| 故事性 | 每章「小事件驱动」弧：问题→应对→落点，不平铺 | 以具体场景/冲突推进，而非概括叙述；一章一钩子一余味 |
| 讲述技巧 | 少「讲」多「演」，用动作/对话/细节承载信息 | show-don't-tell；以孩子（芽芽）的眼折射成人世界 |
| 日常调和 | 硬核种田/战争间，用童真松一口气 | 芽芽的野豆苗、泥马、追水跑，作为节奏阀 |

## 六·八、金句维度 ⭐⭐⭐（用户新增 · 常阳式 · 每章2-3句 · 全文冲5000）

> 常阳的穿越者独白自带现代学科透镜（投资/物理/系统/工程/斯多葛/生物），折射生存本质，aphoristic、反直觉、具象→抽象。目标：每章沉淀2-3句，全文累积至5000句。灵感可借现实阅读（私人储备，见金句库），但**正文中绝不署任何现实人名**——这些话是常阳自己长的。

| 维度 | 要求 | 落点 |
|:---|:---|:---|
| 来源 | 出自常阳的「穿越者独白」，用现代知识解原始困局 | 投资（能力圈/价值机器）、物理（熵增/热力学）、系统（复杂系统呼吸）、工程（渐进解/通用解）、斯多葛（愤怒是奖赏对手）、生物（人与狗的友谊） |
| 形态 | 一句一斩，反直觉重构，具象起、抽象落 | 「工程不是一次画对图纸，是在土里把手磨破、把错改对」 |
| 数量 | 每章 2-3 句，自然不堆砌 | 落位：独白节点 / 章末落点 / 回应小事件后的感悟 |
| 入库 | 每章金句登记进 `故事线追踪/金句库.md`，累计冲5000 | 标注：章节、主题标签、在文中的作用 |
| 禁忌 | 不破坏男频粗粝、不抢戏、不写成说教 | 融进独白与落点，不单列成「名言警句」板块 |

### 灵感储备参考（作者私人 · 源自现实阅读 · 绝不进正文 · 详见 金句库.md「灵感储备」）
1. 投资本质 = 以合适价格买入持续运转的价值创造机器
2. 知道自己能力圈多大，比能力圈有多大重要得多（扩大认知半径，确认能力半径，减少行动半径）
3. 平静是状态，非结果
4. 有选择权时的狠才叫狠；失了选择权的狠，是对无能的羞辱
5. 愤怒是奖励对手的形式（爱比克泰德：激怒你者即你主人）
6. 工程学在不确定里渐进解问题；使命是为某类问题找通用解（大模型=工程学胜利）
7. 解释常是不必要的本能；多数解释可删，理性且回报极高
8. 人和狗的友谊，来自一个莽撞人类与一只温顺野狼
9. 好产品/地点/人，自带某种「传说」
10. 此起彼伏的本质，是复杂系统的生命形态，仿佛呼吸

## 六·九、语体分层与时代演进 ⭐⭐⭐（用户反馈增补 · 沉浸感）

> 用户反馈：现代口语放在文明的上古背景中出戏；语体应随时代演进，慢慢化入当时的口语感（如唐传奇与明清世情小说，语感便已迥异）。核心解法：**双层语体**——现代性只活在穿越者常阳的脑子里，部落族人嘴里只说那个时代的土话。

### 一、双层语体原则（核心）

| 层 | 承载者 | 语体 | 允许的现代性 |
|:---|:---|:---|:---|
| 外层（叙事 + 族人对话） | 部落族人、环境、动作描写 | 上古口语感：拙朴、短句、具体动作、少抽象名词 | 几乎为零。杜绝 玩意儿/搞定/节奏/靠谱/套路/硬核/绝了/牛逼/扯淡/忽悠 等现代口语 |
| 内层（常阳独白 + 系统视角） | 常阳 | 穿越者滤镜：可含游戏术语（面板/尤里卡/科技树/生产力/菜单/进度条）、现代思维、常阳式金句 | 保留。这是"现代人掉进远古"的设定必然，金句生于此层 |

> 判断准绳：**一句话，若是部落族人嘴里说出来的，就不能有现代口语词；若是常阳脑子里转的，可以有。** 例：老猎户说"这玩意儿偷了我半辈子力气"→改"这车偷了我半辈子力气"；常阳想"文明6里点一下车轮就完的玩意儿"→保留。

### 二、时代演进轴（语体随文明换挡）

语体不是一成不变，随"时代进度"从远古走向古典、再往后，口语与叙事的文气逐步加厚——但始终保持"角色说人话、独白带系统"的双层。

| 时代 | 范围 | 外层语体（族人/叙事） | 文气梯度 |
|:---|:---|:---|:---|
| 远古 | 卷一·约 Ch1–Ch30（当前所在） | 最拙朴：短句、具象、靠动作推进；虚词少。代词可用 咱/俺，忌现代连接词与抽象名词 | ★（最口语、最土） |
| 古典 | 卷二起 | 口语略增书卷气：偶现四字格、对仗短句；叙事可引入 城郭/稼穑/百工 等词，但角色对话仍是"说出来的话"，不转文言 | ★★ |
| 中世及以后 | 卷三+ | 继续加厚，可引入更书面词；双层不变 | ★★★ |

> 当前处于「远古→古典」临界（第11章），Ch12 起城址动工、冶炼将启，外层语体可在保持拙朴的同时，随"筑城/百工"悄然添一丝稳重，但不许突然文绉绉。

### 三、现代口语 → 上古口语 替换表

| 现代口语 | 上古/白话替代 | 备注 |
|:---|:---|:---|
| 玩意儿 / 东西（指物） | 物件 / 器物 / 此物 / （具体物名：车、斧、窑） | 族人对话优先用具体物名 |

| 靠谱 | 牢靠 / 稳当 / 靠得住 | |
| 套路 | 路数 / 章法 / 老法子 | |
| 操作 | 作法 / 下手 / 施展 | |
| 绝了 | 妙极 / 神了 / 了得 | |
| 牛逼 | 了得 / 利害 | |

## 六·十、风雅颂与文字出世 ⭐⭐⭐（用户新增 · 诗经融入 · 对应文字科技）

> 用户洞察：故事里应融进《诗经》的风雅颂——既应景，又能体现先人「活泼泼的生命元气」，正好对应「文字出现」这一科技。第一卷大纲已埋 文字系统（Ch63）与 文化爆发·诗歌/历史记录（Ch69），本机制把 Ch69 具象为「华夏诗经诞生」，并以 **oral（口）→ written（文）** 暗线贯通。

### 一、三层映射（风雅颂 → 故事维度）

| 类 | 渊源 | 故事锚点 | 语体层级（见六·九） |
|:---|:---|:---|:---|
| **风** | 民间歌/号子/童谣 | 老阿木伐木号子、夯土号子（Ch13首支种子）、芽芽童谣、梧/石部族歌、巴图带来的异族调 | 远古拙朴口语，**最鲜活的元气**；oral 阶段即可活 |
| **雅** | 城邦正声/宴飨/朝会 | 筑城礼歌、迎宾（巴图）歌、华夏身份歌 | 古典·渐添文气 |
| **颂** | 宗庙/神树/先王 | 伏·神树纪元 神树祭歌、前任首领追颂、开炉/奠基颂、符号石颂（青史线） | 庄重·接古典文气 |

### 二、oral → written 暗线（与科技树咬合）

1. **oral 期（当前·远古→古典临界）**：风以口耳相传活在日常——号子、童谣、宴饮歌。不需文字，正是「活泼泼的生命元气」。近期章节（如 Ch15 巴图登场宴饮）续补民歌种子。
2. **文字解锁（Ch63）**：华夏文字创造。oral 歌第一次有处「落墨」，为诗经编纂备料。
3. **诗经诞生（Ch69，原「文化爆发·诗歌/历史记录」）**：由青史（史官·Ch46登场·文字功底深）**采风编纂**成华夏第一部《诗经》——风采民间、雅定城邦、颂安宗庙；面板跳【文化：诗经 编纂】，给声望/文化值（软实力胜利维度）。伏笔 伏·诗经雏形（歌谣/采诗）于此回收。

### 三、写作落点（与六·八/六·九 协同）

- **风**是六·九「双层语体」外层的最佳载体：族人嘴里唱的，本就该是那个时代的土话歌，绝不用现代口语（呼应六·九替换表）。
- **雅/颂**是「时代演进轴」文气加厚的自然出口：文字出世后，城邦正声与宗庙颂开始带四字格/对仗，正是古典文气（★★）的临界点。
- 一首风可同时承担「人味」（芽芽童谣）、「小高潮」（夯土号子齐喝）、「伏笔」（符号石颂接青史线），一石多鸟。
- **禁忌**：风不可写成文人拟乐府的酸调，要土、要活、要带汗与笑；雅/颂不可抢戏成说文解字。

### 四、首支种子（已埋）

- **Ch13《城墙与房屋》夯土号子**：原文「嗨哟——夯！」定为华夏第一支 oral「风」种子；Ch69 青史采诗时可原句回引，形成「墙里娃笑」的跨章呼应。
- 从第2章（穿越落地）起，严格启用双层语体。

## 六·十一、博采人类文化精华·典籍融入 ⭐⭐⭐（用户升华 · 全书文脉原则）

> 用户升华：「我们是文明，博采人类文化精华」——每章皆可融入历史名著，合情合理、大胆无妨、文采斐然。本原则把六·十的「诗经单点」升为贯穿全书的**文脉机制**。灵魂在「化用」：经典从文明中**自发长出**、引读者共鸣，而非抄袭贴附、更非装逼打脸；正因博采众长、不死守本族线，才见《文明》「合成即进阶」之精髓，也使本书由爽文升华至「故事性 + 可读性俱佳」。

### 〇、灵魂总则（用户定调 · 不可违背）

> 典籍融入的灵魂，不是掉书袋，而是「文明中自发长出来、引发读者共鸣」。

- **化用，不是抄袭**：经典被故事「化」进血肉——情境自然唤起，用故事自己的声音说出，而非原文照搬充门面。读者认出的是共鸣，不是掉书袋。
- **自发长出，不贴上去**：一本书、一句诗，须从人物处境 / 剧情土壤里**自己长出来**；它因「此时此地此人」而恰切，不是作者硬塞的典故。
- **绝不装逼打脸**：典籍**不是**主角碾压他人、秀优越的武器（网文「掉书袋打脸」头号禁用）。它服务于「人」与「境」，不服务于「爽感炫耀」——这正守住男频粗粝与六·九双层语体。
- **博采众长 = 文明游戏精髓**：正因华夏可采众族之长、不必死守本族文化线，才体现《文明》「合成即进阶」的内核——多元熔铸，而非血缘纯正。这是机制的**存在理由**。
- **升华之志**：此机制让书从「爽文」跃为「故事性 + 可读性俱佳」——经典是共鸣的钩子，不是门槛。
- **不点名真实来源（本次修订 · 关键）**：正文与章节元数据里**绝不写出**「老喻」「《诗经·木瓜》」「《孙子》」这类真实人名/书名。句子先在常阳心里**自发涌出**，像土里自己冒出来的；**如必要命名，只在作者侧元数据以「读者向·意近《×》」软注，且一律标「本世界即原创 · 非引经据典」**。读者认出共鸣是好事，但书中人从不"引经据典"——在这个世界，这些话就是他们自己长的。

### 一、核心立意（与设定咬合）

- **穿越即接口**：常阳（伏·穿越谜 穿越者）携全人类经典入世，是华夏吸收人类文化的唯一通道——这把「博采精华」从装饰变成**主题**。
- **与六·八互补**：六·八金句=现代学科透镜（投资/物理/系统…）；六·十一典籍=人文经典透镜（诗/骚/老/庄/孙/史/荷马/圣经…）。一硬一软，共铸常阳独白。
- **与六·九不冲突**：典籍是「文」不是「现代口语」，反而比现代 slang 更贴上古语境；关键是落在正确层级（见下）。
- **与灵魂总则同源**：本机制的「存在理由」是文明游戏的精髓（博采众长、不为本族线所缚），其「质量志」是把书由网文升格为故事性与可读性俱佳（见〇）。

### 二、三层落点（严守双层语体）

| 层 | 可融入的典籍 | 示例 | 红线 |
|:---|:---|:---|:---|
| **内层（常阳独白）** | **任意人史经典**，作穿越者文化透镜 | 以《孙子》度战、以《道德经》解「无为而治」、以荷马喻远征、以《史记》笔法记人 | 现代脑中，六·九放行 |
| **外层·自产（古典起）** | 华夏自己长出的典籍（诗经→诸子→史书） | Ch69 后族人可引《诗经》；更后可引诸子 | 只引「时代够得着」的自产经典，不提前塞 |
| **外层·oral/folk** | 谚语、格言、劳动号子、歌谣（风） | 远古用谚语/号子；任何时代都自然 | 接六·十，要土要活 |

### 三、合情合理红线（用户原话）

- **合情合理，大胆使用无妨**：经典须「化用」而非「掉书袋」——融进情绪/场景/动作，不单列成「名言警句」板块。
- **不破男频粗粝**：宁可少而精，一章 1–2 处足矣；宁缺毋滥，不为掉书袋牺牲热血节奏。
- **不破双层语体**：族人嘴里不背跨时代/跨文明的硬经典（除非是华夏自产且时代够得着）；异文化经典走常阳独白或「转述为华夏人话」。
- **真实名著**：用人类真实遗产（诗经/老子/荷马/圣经/一千零一夜…），不杜撰伪典（华夏自产典籍如《诗经》属 in-world 真经典）。
- **绝不装逼打脸（见灵魂总则）**：典籍服务于「人」与「境」，绝不用于主角秀优越 / 碾压他人；违此即背离六·十一灵魂，亦破男频粗粝。
- **正文绝不点名真实典籍（本次修订）**：共鸣句在角色心里先自发涌出；章节元数据如需追踪，以「读者向·意近《×》」软注并标「本世界即原创·非引典」，绝不写进叙事让人物"引经据典"（见〇·不点名真实来源）。

### 四、转动菜单（避免偏食）

- 建 `故事线追踪/典籍素材库.md`：按文明/时代分类的真实名著 + 可化用场景 + 示例短句，逐章轮用，确保「博采」而非偏嗜一隅。
- 中国线：诗/骚/老/庄/孟/孙/史/汉赋/唐诗/宋词/红楼…；世界线：荷马/圣经/吉尔伽美什/吠陀/塔木德/一千零一夜/莎士比亚/启示录…
- 越往后，华夏自产典籍越丰（诗经Ch69→诸子→史书），外层可引面渐宽；常阳独白则始终可引任意时代经典。

### 五、元数据建议

## 六·十二、文明更替与历史真髓（翦商精神）⭐⭐⭐

> 你的故事逐渐触及《翦商》的内核——文明大更替的时刻：一种以人祭/恐怖凝聚的旧秩序，被一种以"德"与"天命"凝聚的新秩序替代。这不是历史复刻，而是文明周期律的隐喻。以下为写作原则。

| 维度 | 原则 | 执行 |
|:---|:---|:---|
| 旧秩序具象化 | 炎部落/铜甲大部/弯钩记号（〆）= 旧纪元人祭制度的残余。青铜/铜器+神秘祭仪+中央神权+封闭掠夺+以恐怖凝聚 | 写作中不直呼"人祭"，以细节暗示（弯钩记号、碎骨片、废弃祭坛、战俘待遇） |
| 新秩序具象化 | 华夏·阳城 = 新秩序萌芽：铁器+贸易+开门+驯化+以共同利益凝聚 | 常阳的决策自然体现"以德服人"、"天命在民"，但不使用这些标签——让道路的差异自己说话 |
| 常阳的觉醒 | 他逐渐意识到自己面对的不仅是军事对手，而是一种**文明道路的选择**：生产性秩序 vs 祭祀性秩序 | 内心独白从"怎么赢"逐步深化为"怎么活"——不是他一个人在打仗，是一个文明在选路 |
| 天命的自然涌现 | "天命"不是神赐，是民心与生产力之和——文明因"让人活得更好"而获得天命，因"吃人/耗尽人"而失去天命 | 面板数据（声望/人口/生产力）可视为天命的量化侧面；常阳的抉择参考"这样做，人活得更像人吗？" |
| 不点名铁律 | 正文中绝不出现「翦商」「商」「周」「人祭」「天命」「德治」等历史标签 | 这些是魂，不是壳；所有历史真髓化为情节与细节，读者认出共鸣是好事，但书中人不引经据典 |
| 与六·八/六·十一协同 | 六·八金句=常阳的文明洞察；六·十一典籍=文明的文化透镜；六·十二=文明的深层驱动力 | 三者合一，金句可暗扣道路抉择，典籍可暗含历史真髓，但都在"自发涌出·本世界即原创"的框架内 |

**核心心法（写每章前默念）**：旧秩序因"吃人"而亡，新秩序因"让人活得更好"而兴——这不是道德评判，是文明周期的铁律。常阳和华夏走的每一步，都在无声地选择自己要成为哪一种文明。

## 六·十三、吟游诗人尾诗 ⭐⭐⭐（用户新增 · 章节尾史诗吟诵 · 恢弘历史感）

| 维度 | 要求 | 落点 |
|:---|:---|:---|

## 六·十四、章节标题统一规则 ⭐⭐⭐（用户新增 · 标题体式）

| 维度 | 要求 | 反例→正例 |
|:---|:---|:---|
| 标点 | **禁止逗号分割**，标题为单一短语或短句，不用「，」 | ❌「草创之时，文明抽芽」→ ✅「草创之时文明抽芽」 |

### 命名指引

- **留白**：不强求工整对仗，长短错落本身就是质感。

## 六·十八、章节篇幅与丰满度 ⭐⭐⭐（用户新增 · 每章约3000字 · 丰盈可读性）

> 用户升华：现有每章约 1500–2000 字，文风过于极简干冽，缺乏环境背景描写、有趣对话与深度思考。要求**每章扩至约 3000 字（纯汉字正文，不含元数据；尾诗另计）**，在**不新增情节**的前提下变得丰满丰盈、可读性再提升。理由：①起点每章 3000 字惯例；②补足环境/对话/思考三层质感。

| 维度 | 要求 | 落点 |
|:---|:---|:---|
| 篇幅 | **每章正文约 3000 字**（下限 2800，上限 3300）；尾诗、元数据不计入 | 取代原 1800–2200 目标（见 README/大纲） |
| 不增情节 | **严格不新增事件、场景、人物、伏笔钩子**；只在同一情节骨架上补肉 | 既有场景节拍、出场人物、章末钩子全部保留；扩写=同场景内加细节，非加戏 |
| 环境背景 | 补足感官质感：光/声/嗅/温/湿/材质/天候/地貌；让世界可触可闻 | 例：火塘烟的走向、兽皮的刮感、平原热浪、河面碎光、远山轮廓 |
| 有趣对话 | 加重自然角色对话（常阳 gamer 吐槽/冷幽默、大壮粗粝、阿陶专注、巴图商贩腔、老石头固执、芽芽童音），以对话显性格、带气氛，忌信息倾倒 | 对话服务人物与氛围，不替代叙事；口语化、有节奏 |
| 深度思考 | 加重常阳 internal monologue：战略推演、文明/翦商洞察、领导力重量、"这像游戏却真要人命"的张力、自我怀疑 | 思考贴情节、由场景触发，不空发议论 |
| 文风锚 | 仍守**男频·热血粗粝 + 稍带口语化**；丰盈≠软塌，细节要硬朗有劲 | 与六·九 双层语体协同：现代性只在常阳内心；部落言语=上古口语 |
| 与硬规则协同 | 扩写后仍需 **0「不是…是」**（六·一）· 不点名现实书名（六·十一）· 尾诗保留（六·十三/十五） | 扩写完毕须过 `不是.{0,20}是` 正则复核 |

### 操作守则（扩写既有章）
- 读原章 → 锁定情节骨架（场景列表/出场人物/章末钩子）→ 在骨架内逐场景加环境/对话/思考 → 保留 `> 🎵` 尾诗块逐字 → 更新 `📊 章节元数据` 的「实际字数」→ 跑 `不是.{0,20}是` 复核。
- 卷一 36 章统一扩写；后续新章直接按 3000 字写，不必二次扩。

## 六·十九、创作引导与大纲关系 ⭐⭐⭐（用户新增 · 大纲动态调整 · 即兴优先）

> 用户原则：创作中即兴加入的指引若与大纲冲突，**动态调整大纲**——可延后、可重写、不必削足适履。大纲是辅助框架，不是铁律；当下感受与灵感优先于预设规划。小说因此保有个人风格与独特节奏。

| 维度 | 要求 | 落点 |
|:---|:---|:---|
| 大纲角色 | 大纲是**活文档**（living document），随创作即时调整 | 不预设"大纲不可改"；即兴灵感→更新大纲，而非压抑灵感迁就大纲 |
| 冲突处理 | 实写与大纲不符时，**优先保留实写**，回头对齐大纲 | 已有实例：阶段四原规划战争弧→实写为结绳/文字/吟游传说弧，已按实写重写大纲 |
| 调整方式 | 可延后（把原大纲章节推后）、可重写（全部替换）、可删改 | 不保留"待统一"标记——即刻修改 |
| 版本记录 | 大纲的每次重大调整在 git log 中可追溯；不刻意保留旧版本标记 | 大纲是当下的导航，不是历史的档案 |

### 与设定协同
- 大纲 `第一卷大纲.md` 阶段四已按实写重写（Ch31–36 结绳/文字/吟游弧）；阶段五起按新轨迹调整。
- 此原则体现于"用户即兴→落地为六·规则→大纲动态对齐"的工作流闭环：用户指令→记录为六·规则→大纲随之调整→正文产出。大纲永远跟随创作走。

## 六·二十、跨章叙事与戏剧张力 ⭐⭐⭐（卷一后半程新增 · Ch67+）

> 用户反馈：已完成章节"平实有余，戏剧张力不足"——每章是独立小故事，缺乏跨越多章的长线谜题。自Ch67起，叙事驱动力从"生存建设"转向"真相探索"，以三个长线谜题（面板之源/跨界者/坛城崩塌）替代"逐一解决问题"模式。

### 一、章首钩子强制原则

每章前100字必须出现一个"为什么"——让读者带着问题进入本章。**禁止**以"XXX的消息是XX带回来的"或"XXX之后的一天"这类平铺告知式开头。

| ❌ 平铺告知 | ✅ 钩子驱动 |
|:---|:---|
| "废石山出事的消息是阿雅带回来的。" | "阿雅回来的时候，我看了一眼她的脸，就知道废石山已经不在了。" |
| "第七块石头立起来那天，北山石鼓的人来看了一眼。" | "北山的人在天亮前就到了。他们不是来买东西的——是专程来看石头的。" |
| "阿依走后留下了一卷羊皮。" | "羊皮卷上的符号不是云中部的文字。我认得其中几个——那是文明6的奇观图标。" |

### 二、信息滞后原则（延迟揭示）

关键信息不一次性揭示，延后3-5章徐徐展开。每个谜题从"暗示"到"确认"到"完全揭示"至少跨越三章。

| 谜题 | 暗示（Ch n） | 确认（Ch n+2） | 揭示（Ch n+5） |
|:---|:---|:---|:---|
| 阿依的秘密 | Ch67羊皮卷奇观符号 | Ch74周期之壁对应 | 仅暗示不点明（留白） |
| 坛城起源 | Ch79阿依骑兵南下 | Ch87坛城使者到来 | Ch90坛城领袖密信 |
| 前任持有者 | Ch70面板日志·第47条 | Ch85日志更新"所有的路尽头都一样" | Ch100世界议会空柱 |
| 观察者降临 | Ch90面板倒计时 | Ch94"已进入大气层" | Ch100议会空间 |

### 三、章末跨章钩子强制原则

每章最后一句必须指向下一章——可以是悬念、问题、异常信号或不可解释的现象。**禁止**以"这一夜过去了"或"明日又是新的一天"这类收束式结尾。

| ❌ 封闭式结尾 | ✅ 钩子式结尾 |
|:---|:---|
| "阳城的日子，就这么一天天过着。" | "常阳在马背上收到面板信息——'前任持有者日志·第47条'。" |
| "明天又是新的一天。" | "地平线上出现了不属于任何已知部落的火光。" |

### 四、节奏律动

- 每5章一个小高潮（揭示/反转/战斗/情感爆点）
- 每10章一个大高潮（谜题阶段性解答）
- 阶段钩子型章（第70/80/90/100章）：必须包含"下一阶段核心谜题"的第一次正面揭示
- 静默章（节奏缓冲）：在两个高潮之间插入1章"呼吸空间"——写日常、写感情、写风景，但章末钩子不能丢

### 五、谜题驱动而非事件驱动

每次动笔前，先问："这一章要让哪一个谜题前进一小步？"

| 旧模式 | 新模式 |
|:---|:---|
| 阳城遇到了什么问题？→解决它 | 哪一个谜题到了该推进的节点？→让它前进 |
| 这一章要写完一件完整的事 | 这一章要打开一扇门，下一章走进去 |
| 章末：事情解决了，收束 | 章末：谜题靠近了，但还没完全解开 |

## 七、风格样本段落 ⭐

> 以下为风格锚点，正文生成时应与此一致。

---

我揉了揉眼睛，以为自己还在工位上做了个离谱的梦。

直到一个满脸褶子的老头凑到我面前，用我听不懂但莫名能理解的语言说：「首领，您醒了。」

首领？

我低头看了看自己——格子衬衫不知道什么时候变成了兽皮，眼镜没了，手腕上多了一道环状印记，正微微发着光。

然后，半透明的蓝色面板在我眼前展开。

【文明面板】
【文明：华夏】
【时代：远古】
【人口：47】
【粮食：3天】
【科技：石器时代】

我愣了三秒。

这界面……怎么跟文明6一个模子刻出来的？

「等等，」我下意识摸了摸手腕，「让我看看面板……」

老族长以为我在召唤神灵，立刻跪下了。

---
