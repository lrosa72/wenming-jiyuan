## 📖 故事简介

程序员常阳加班猝死，醒来发现自己穿越成了石器时代一个原始部落的首领——四十七人、三天存粮、石器武装。更糟的是，蛮族大军已经压境。

他手里唯一的依仗，是一块半透明的蓝色「文明面板」——像文明6的游戏界面，但每一行数据背后都是真人的生死。

这不是游戏。饿死人不会重开，死掉的战士不会在下一回合复活。

从石器到铜器，从铜器到铁器；从一座土城到一张贸易网络，从一个人到一百人、再到一个文明——

常阳必须在这片盖亚大陆上，带着华夏部落活下来。而大陆上有七座文明之城，其中一座正在吞食其他六座——南方的「坛城」，一个以人祭维系秩序的旧时代怪物。

这是一场文明道路的选择：恐惧与生产，祭祀与贸易，旧秩序与新秩序。华夏的每一步，都在无声地决定自己要成为哪一种文明。

## 🌊 故事内核：翦商精神

本书的深层脉络，与《翦商》的内核共鸣：

> **旧秩序因「吃人」而亡，新秩序因「让人活得更好」而兴。**
> 这不是道德评判，是文明周期的铁律。

书中的两条道路：
- **坛城之路**：青铜+人祭+恐惧+封闭——以恐怖凝聚，靠吃人生存
- **华夏之路**：铁器+贸易+生产+开放——以共同利益凝聚，让人活得更像人

每一条金句、每一场冲突，都在无声地回应这个主题。

## 🧩 五大写作原则

### 1. 摸石过河（非游戏攻略）
文明的发展 **不是**「规划→解锁→完成」的线性流程。
- 铁犁是试错三块废铁后磕磕绊绊诞生的
- 分工是模糊的——谁有空谁干，没有定岗
- 面板不施舍，只照见——它只是记录下笨拙而真实的人类努力

### 2. 金句两品（累计 63+ 条 / 目标 5000）
- **原创金句**：常阳心里自长的aphorism，现代学科透镜折射原始生存
- **应用名句**：典籍名句在角色心中自发涌出，**不点名真实来源，本世界即原创**

> 铁律：正文中绝不署任何现实人名/书名；所有句子在角色心里自发涌出，读者认出共鸣是好事，但书中人从不「引经据典」。

### 3. 人味大于设定（芽芽与灰灰原则）
文明六的游戏机制是骨架，人的温度才是血肉。
- 芽芽驯化灰灰（一只野狼崽变成第一条家犬）的过程，是全书驯化线的起点
- **GGS驯化法则**（《枪炮、病菌与钢铁》）：可驯化的大型哺乳动物仅十余种，芽芽的「通兽亲和」只对此生效——非万兽皆可收

### 4. 血腥不回避（坛城的阴影）
旧秩序的残酷通过逃出者的眼睛呈现——不渲染、不煽情，用最平的叙述写最重的事：
- 铜棍落下时「像石头砸进烂泥」的声音
- 祭路上一里地二十三个小土堆
- 炎部落老铁匠打了二十年铁留不住一把刀（「铁是祭坛的。人也是。」）

### 5. 外部压力同步增长
阳城每长一分，坛城的注意力就多一分。内外夹逼才是文明的常态。
- 「阳城长了，烟也粗了。没这么巧。」

