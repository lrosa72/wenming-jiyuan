# 人物设定 — 芽芽

> 常驻暖色角色（第7章登场），用于调和日常、注入人味。

---

## 基本档案

| 维度 | 内容 |
|:---|:---|
| 姓名 | 芽芽（小名，本名未明） |
| 年龄 | 约 6 岁 |
| 身份 | 华夏部落孩童（孤儿，原属被蛮族劫掠的小部落） |
| 登场 | 第7章（人口繁衍） |
| 定位 | 日常暖色 / 童真视角 / 节奏阀 |

---

## 外貌与性格

- 瘦小，麻花辫常散一根，鼻尖总沾泥。
- 不怕生，不认怂。头回见常阳，不哭不躲，仰脸问"你就是那个让地长粮食的头人？"
- 对活物有痴劲（守野豆苗能半天不动，虫子爬手背不拂），这份痴像极了烧第一只盆时的阿陶。
- 是部落里最不把常阳当首领的人——往他小腿上撞、揉乱他头发、叫他"头人"。

---

## 关系

| 关系人 | 关系 | 说明 |
|:---|:---|:---|
| 常阳 | 依赖/崇拜（不叫首领，叫头人） | 常阳蹲下与她平视，是她心里"最好的人" |
| 阿陶 | 黏/师徒味 | 认准阿陶，天天蹲窑边；阿陶给她捏歪脖子小泥马 |
| 老石头 | 嘴硬心软的爷爷 | 芽芽喊"石头爷爷大方"，老石头耳根红 |
| 石部落孩童 | 玩伴 | 来投首日被她塞薯干破冰 |

---

## 戏用方向（伏笔/功能）

- **人味阀**：硬核种田/战争章之间，用她的野豆苗、泥马、追水跑松节奏。
- **视角折射**：以孩子眼折射成人世界的紧张（敌意、融合、远忧）。
- **未来钩子**：可成长为华夏第一代"在和平里长大"的孩子，对照常阳"穿越者"身份；她的野豆苗可象征华夏生根。
- 不参与战斗线，保持纯真，是常阳"为谁而守"的情感锚之一。

## 特殊能力：通兽亲和（第16章激活）

| 维度 | 内容 |
|:---|:---|
| 能力名称 | 通兽亲和（盖亚驯化法则·天授） |
| 首次激活 | 第16章（坡地救灰毛狼崽，母狼弃崽、芽芽以手触之即降服） |
| 首只驯化兽 | 灰灰（灰毛狼崽→华夏首犬，伴生驯化兽，第16章命名） |
| 能力本质 | 对"肯向人类低头的稀有兽种"具有天生亲和力——兽不咬她，反往她怀里拱。老族长点破："老辈传下话，有种人天生通兽，兽不咬她。" |
| 能力边界（GGS驯化法则·硬约束） | **此能力绝非万兽皆可收**。盖亚大地上兽类万千，但能被人类驯化的大型哺乳动物屈指可数（史上仅十余种）——狗（自狼来）、牛、羊、猪、马……此为数种外，其余万兽野性难驯、世代不可降。芽芽的通兽亲和**只对这寥寥几种"可驯之物"生效**：狼崽（→狗）是她的第一头，未来或可收山羊/绵羊/牛犊，但对虎豹熊罴之类绝无效力——那些是"不可驯之物"，她碰了也会被咬。 |
| 与大壮驯马的区别 | 大壮驯马属役用驯化（暴力+耐心+半月降服），芽芽驯犬属伴生驯化（亲和+共情+一刻钟入怀）。二者分属不同驯化线，不互斥。马=脚力/战力，狗=伴儿/嗅探/警戒。 |
| 叙事功能 | ①人味阀：灰灰是芽芽的影子，日常场景的暖色锚点；②嗅探辅助：灰灰嗅觉为探矿/追踪提供便利（Ch17 废石山嗅探铁矿石）；③情感对照：芽芽是孤儿，灰灰也是孤儿，两个小东西凑一对——"拢了野狼崽也拢了人心"。 |
| 未来成长 | 灰灰从幼崽长成成年犬（警戒/放牧/狩猎辅助）；芽芽或可再收一头羊/牛犊（GGS约束内），但绝不会变成"百兽之王"式的童话设定——GGS驯化法则即为此设的硬天花板。
