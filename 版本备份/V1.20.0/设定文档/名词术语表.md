## 一、核心系统术语（文明面板）

| 术语 | 含义 |
|:---|:---|
| 文明面板 | 常阳的金手指，可视化显示文明各项数据的半透明界面 |
| 科技树 | 科技解锁的路径树，按依赖关系逐级解锁 |
| 政策树 / 政策卡 | 政体下的政策系统，每时代可更换 |
| 尤里卡 | 面板机制：完成特定现实行为可加速科技研发（省10%进度） |
| 时代加成 | 进入新时代后获得的全局buff（如远古+10%粮食） |
| 奇观 | 全球唯一的特殊建筑，先建成者独占，与两界通道相关 |
| 战略资源 | 马匹、铁矿等影响兵种解锁的关键资源 |
| 国力评级 | 面板综合评估（F→A→S逐级） |
| 世界议会 | 远古时代结束后逐步解锁的高级机制（第20章预告） |
