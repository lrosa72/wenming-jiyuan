# 写作风格指南 — 《天地为棋，文明纪元》

> 本文件为 S03（正文生成）和 S05（内容打磨）提供风格基准。

---

## 一、叙事视角

**第一人称** — 常阳视角，严格使用「我」。不允许随意切换到第三人称。

- 读者通过常阳的眼睛看世界
- 常阳不知道的信息，读者也不知道（除非通过面板提示）
- 内心独白是重要组成部分（游戏玩家的吐槽、分析、算计）

---

## 二、语言风格

**男频主调：热血粗粝 + 稍带口语化 + 现代游戏玩家腔**

> 基调转向男频——节奏狠、感官糙、带点嘴上不干净的真性情。猫腻式的「凝练白描」保留为**写作技法**（见六·六），但不再是整体语气主锚。

| 特征 | 说明 |
|:---|:---|
| 节奏 | 快、狠、短。段落 2-4 行，战斗章可 1-2 行一句，手机阅读如刀 |
| 语气 | 口语化、带劲、稍带粗口（他娘的/老子/这帮），但「稍带」不泛滥 |
| 热血 | 推进靠冲劲与胜负欲，主角算计里裹着狠劲，不温吞 |
| 粗粝 | 多写血汗土铁腥，少写文艺比喻；用硬物象，不铺柔光 |
| 幽默 | 现代社畜吐槽保留，但收着，严肃章直接掐 |

---

## 三、对话风格

| 角色 | 说话风格 |
|:---|:---|
| 常阳 | 现代口语，带游戏术语（「这波尤里卡太关键」「要卡科技树了」），思考时自言自语 |
| 老族长 | 古朴稳重，多用比喻和部落谚语 |
| 大壮 | 简短有力，很少废话，行动派 |
| 阿陶 |  excited，经常「哦！我懂了！」 |
| 巴图 | 圆滑世故，商人腔调，爱说「在我的旅途中……」 |

**对话占比**：30-40%

---

## 四、描写密度

| 类型 | 比例 | 说明 |
|:---|:---|:---|
| 环境描写 | 15% | 2-3句精准感官细节，服务于氛围 |
| 心理描写 | 25% | 主角的分析、算计、吐槽 |
| 动作描写 | 30% | 战斗、建设、操作面板 |
| 对话 | 30% | 推进剧情和展示性格 |

---

## 五、节奏偏好

- **快节奏开场**：前300字必须进入状态（冲突/悬念/动作）
- **章末留钩子**：不让读者有「可以停下来」的感觉
- **爽点密度**：每章至少1个小爽点（面板数据暴涨/科技突破/敌人吃瘪）
- **长短句交替**：刻意制造长短句变化，避免AI味的对称结构

## 六、参考作品（文风锚）

**主锚（男频 · 热血粗粝 · 口语化）：**
1. **《雪中悍刀行》**（热血+粗粝+口语，那种"老子就是要扛"的劲）
2. **《放开那个女巫》**（种田流建设爽感，男频干净利落的叙述）
3. **《临高启明》/《伐清》**（硬核种田，技术落地的解气感）
4. **《夜的命名术》**（现代口语+热血，主角算计与冲劲并存）

**技法参考（白描/凝练，仅作手艺，不当语气）：**
5. **猫腻作品**（《庆余年》《间客》《将夜》）— 取其「凝练白描、短句呼吸」，不取其克制文艺的整调
6. **《修真四万年》**（科技感+热血进化流）
7. 文明系列游戏文本（civilopedia 的考究感）

## 六·五、句式硬规则 ⭐⭐⭐（最高优先级）

> **彻底禁用「不是……，是……」句式。**

沿用前序「炉石小说」项目的明确规则，无任何例外。

### 禁止形式

| 形式 | 示例 | 违规 |
|:---|:---|:---|
| 标准式 | 「不是椅子，是泥地」 | ❌ |
| 并列式 | 「不是空调味，也不是酸臭味」 | ❌ |
| 拆分式 | 「不是普通的闪烁。是一道光斑。」 | ❌ |
| 因果式 | 「不是因为胆子大，是因为……」 | ❌ |

### 修正方法

用**白描 / 直接陈述 / 意象对比**替代，绝不出现「不是」与「是」的对应结构。

| 违规 | 修正 |
|:---|:---|
| 不是椅子，是泥地 | 身下是铺着兽皮的硬泥地 |
| 不是普通的闪烁。是一道光斑 | 屏幕炸开一团白光 |
| 不是因为胆子大，是因为…… | 胆子不大。脑子里那根弦却转得飞快 |

### 自检清单（每章必查）

- [ ] 全文搜索「不是」，确认无「不是X，是Y」结构
- [ ] 全文搜索「不是」+「是」，确认无跨句拆分变体
- [ ] 全文搜索「也不是」「也不是Y」，确认无并列变体

## 六·六、男频文风主锚 ⭐⭐⭐（热血粗粝 · 口语化）

整体语气走男频：冲、糙、带点嘴上不干净的真性情。猫腻的「凝练/白描/短句」降为**手艺**，融进男频骨架里，不取其温吞文艺的整调。

| 维度 | 男频表现 | 本章落点 |
|:---|:---|:---|
| 热血冲劲 | 推进靠胜负欲与狠劲，主角算计里裹着冲 | 战斗章句短如刀，胜了要爽到 |
| 粗粝感官 | 多写血/汗/土/铁锈/骨，硬物象，不铺柔光 | 落石、火、血溅用硬词 |
| 稍带口语 | 他娘的/老子/这帮/哥几个，克制不泛滥 | 常阳内心与短吼用口语 |
| 白描（手艺） | 用动作/物象说话，不直述情绪 | 「恐惧」用族人低头写 |
| 短句呼吸 | 三两行一断，战斗可一句一行 | 段落长短有杀气 |
| 意象克制 | 以具象代抽象，但意象要硬 | 面板红光→「血一样的红漫上来」 |
| 不滥情 | 悲壮暗写，不点破，不矫情 | 前一任首领之谜用老族长眼神 |
| 收幽默 | 现代吐槽保留但收，严肃章直接掐 | 战争/牺牲章不抖机灵 |

### 普通写法 → 男频式（热血粗粝 + 白描）

| 普通写法 | 男频式 |
|:---|:---|
| 他感到很恐惧 | 他低了头，手指抠进泥里，指节发白 |
| 这让他非常震惊 | 他愣了片刻，瞳孔缩成针尖 |
| 常阳心里想：这太荒谬了 | 常阳盯着腕上的光，半天没蹦出一个字 |
| 环境很荒凉 | 风卷黄土，山脉的尽头看不见 |
| 他赢了，很高兴 | 他抹了把脸上的血，咧嘴，牙全是红的 |
| 蛮族很可怕 | 那帮蛮子眼珠子发绿，像饿了一冬的狼 |

## 六·七、人味与故事性 ⭐⭐（用户反馈增补）

> 用户指出：文章流畅但缺点人味、故事性不强、讲述技巧平淡。修订方向如下。

| 维度 | 要求 | 落点 |
|:---|:---|:---|
| 人味 | 角色像活人，有软处、有温度 | 引入常驻暖色角色（芽芽·六岁孤女）；常阳冷硬外壳下的软；族人小互动（老石头藏肉干、阿陶捏泥马） |
| 故事性 | 每章「小事件驱动」弧：问题→应对→落点，不平铺 | 以具体场景/冲突推进，而非概括叙述；一章一钩子一余味 |
| 讲述技巧 | 少「讲」多「演」，用动作/对话/细节承载信息 | show-don't-tell；以孩子（芽芽）的眼折射成人世界 |
| 日常调和 | 硬核种田/战争间，用童真松一口气 | 芽芽的野豆苗、泥马、追水跑，作为节奏阀 |

## 六·八、金句维度 ⭐⭐⭐（用户新增 · 老喻风 · 每章2-3句 · 全文冲5000）

> 用户偏好老喻《人生算法》式金句——以现代学科透镜（投资/物理/系统/工程/斯多葛/生物）折射生存本质，aphoristic、反直觉、具象→抽象。目标：每章沉淀2-3句，全文累积至5000句。

| 维度 | 要求 | 落点 |
|:---|:---|:---|
| 来源 | 出自常阳的「穿越者独白」，用现代知识解原始困局 | 投资（能力圈/价值机器）、物理（熵增/热力学）、系统（复杂系统呼吸）、工程（渐进解/通用解）、斯多葛（愤怒是奖赏对手）、生物（人与狗的友谊） |
| 形态 | 一句一斩，反直觉重构，具象起、抽象落 | 「工程不是一次画对图纸，是在土里把手磨破、把错改对」 |
| 数量 | 每章 2-3 句，自然不堆砌 | 落位：独白节点 / 章末落点 / 回应小事件后的感悟 |
| 入库 | 每章金句登记进 `故事线追踪/金句库.md`，累计冲5000 | 标注：章节、主题标签、在文中的作用 |
| 禁忌 | 不破坏男频粗粝、不抢戏、不写成说教 | 融进独白与落点，不单列成「名言警句」板块 |

### 老喻金句参考（已收集，详见 金句库.md「参考素材」）
1. 投资本质 = 以合适价格买入持续运转的价值创造机器
2. 知道自己能力圈多大，比能力圈有多大重要得多（扩大认知半径，确认能力半径，减少行动半径）
3. 平静是状态，非结果
4. 有选择权时的狠才叫狠；失了选择权的狠，是对无能的羞辱
5. 愤怒是奖励对手的形式（爱比克泰德：激怒你者即你主人）
6. 工程学在不确定里渐进解问题；使命是为某类问题找通用解（大模型=工程学胜利）
7. 解释常是不必要的本能；多数解释可删，理性且回报极高
8. 人和狗的友谊，来自一个莽撞人类与一只温顺野狼
9. 好产品/地点/人，自带某种「传说」
10. 此起彼伏的本质，是复杂系统的生命形态，仿佛呼吸

## 六·九、语体分层与时代演进 ⭐⭐⭐（用户反馈增补 · 沉浸感）

> 用户反馈：现代口语放在文明的上古背景中出戏；语体应随时代演进，慢慢化入当时的口语感（如唐传奇与明清世情小说，语感便已迥异）。核心解法：**双层语体**——现代性只活在穿越者常阳的脑子里，部落族人嘴里只说那个时代的土话。

### 一、双层语体原则（核心）

| 层 | 承载者 | 语体 | 允许的现代性 |
|:---|:---|:---|:---|
| 外层（叙事 + 族人对话） | 部落族人、环境、动作描写 | 上古口语感：拙朴、短句、具体动作、少抽象名词 | 几乎为零。杜绝 玩意儿/搞定/节奏/靠谱/套路/硬核/绝了/牛逼/扯淡/忽悠 等现代口语 |
| 内层（常阳独白 + 系统视角） | 常阳 | 穿越者滤镜：可含游戏术语（面板/尤里卡/科技树/生产力/菜单/进度条）、现代思维、老喻金句 | 保留。这是"现代人掉进远古"的设定必然，金句生于此层 |

> 判断准绳：**一句话，若是部落族人嘴里说出来的，就不能有现代口语词；若是常阳脑子里转的，可以有。** 例：老猎户说"这玩意儿偷了我半辈子力气"→改"这车偷了我半辈子力气"；常阳想"文明6里点一下车轮就完的玩意儿"→保留。

### 二、时代演进轴（语体随文明换挡）

语体不是一成不变，随"时代进度"从远古走向古典、再往后，口语与叙事的文气逐步加厚——但始终保持"角色说人话、独白带系统"的双层。

| 时代 | 范围 | 外层语体（族人/叙事） | 文气梯度 |
|:---|:---|:---|:---|
| 远古 | 卷一·约 Ch1–Ch30（当前所在） | 最拙朴：短句、具象、靠动作推进；虚词少。代词可用 咱/俺，忌现代连接词与抽象名词 | ★（最口语、最土） |
| 古典 | 卷二起 | 口语略增书卷气：偶现四字格、对仗短句；叙事可引入 城郭/稼穑/百工 等词，但角色对话仍是"说出来的话"，不转文言 | ★★ |
| 中世及以后 | 卷三+ | 继续加厚，可引入更书面词；双层不变 | ★★★ |

> 当前处于「远古→古典」临界（第11章），Ch12 起城址动工、冶炼将启，外层语体可在保持拙朴的同时，随"筑城/百工"悄然添一丝稳重，但不许突然文绉绉。

### 三、现代口语 → 上古口语 替换表

| 现代口语 | 上古/白话替代 | 备注 |
|:---|:---|:---|
| 玩意儿 / 东西（指物） | 物件 / 器物 / 此物 / （具体物名：车、斧、窑） | 族人对话优先用具体物名 |
| 家伙（指工具/兵器） | 器物 / 兵刃 / 物件 | |
| 搞定 | 办成 / 了了 / 成事 | |
| 节奏 | 删抽象，写动作 / 缓急 / 章法 | |
| 靠谱 | 牢靠 / 稳当 / 靠得住 | |
| 套路 | 路数 / 章法 / 老法子 | |
| 操作 | 作法 / 下手 / 施展 | |
| 绝了 | 妙极 / 神了 / 了得 | |
| 牛逼 | 了得 / 利害 | |
| 扯淡 | 胡吣 / 瞎说 | |
| 忽悠 | 诓 / 哄骗 | |
| 根本（副词） | 全 / 浑 / 竟 | "根本不怕死"→"全不怕死" |
| 其实 | 实则 / 说到底 / 删 | |

### 四、序章例外

- **第1章（穿越前序章）**：设定在现代办公室，中央空调/钉钉/泡面/套路 等现代词**保留**，这是时代背景本身。
- 从第2章（穿越落地）起，严格启用双层语体。

## 七、风格样本段落 ⭐

> 以下为风格锚点，正文生成时应与此一致。

---

我揉了揉眼睛，以为自己还在工位上做了个离谱的梦。

直到一个满脸褶子的老头凑到我面前，用我听不懂但莫名能理解的语言说：「首领，您醒了。」

首领？

我低头看了看自己——格子衬衫不知道什么时候变成了兽皮，眼镜没了，手腕上多了一道环状印记，正微微发着光。

然后，半透明的蓝色面板在我眼前展开。

【文明面板】
【文明：华夏】
【时代：远古】
【人口：47】
【粮食：3天】
【科技：石器时代】

我愣了三秒。

这界面……怎么跟文明6一个模子刻出来的？

「等等，」我下意识摸了摸手腕，「让我看看面板……」

老族长以为我在召唤神灵，立刻跪下了。

---

*注：以上样本展示了——第一人称视角、现代口语、游戏术语自然融入、内心吐槽、面板可视化的爽感、短段落节奏。*
