# 第5章 部落之名

胜仗过后第三天，谷里总算有了活气。

伤的上了草药，断弓重新绷弦。那两个没回来的，埋在坡后头，土堆前插了截石斧。没人多哭，部落的规矩——人死了，活着的人得把日子往下过。我站坡上往下看，四十七个人，如今四十五个，可腰杆比从前直。我深吸一口气，风里有血腥退下去的土味，也有活人的热。赢了仗，粮有了，盆有了，马也在盯着的路上，可人心是飘的，得有个东西拴住。

我沿着坡走了一圈。有人在补兽皮，有人教娃认箭簇，阿陶在窑边擦他那些盆。赢了之后，人人都在忙，可忙得散。没人说"咱们为啥忙"。我从前觉得，活下来就行；如今懂了，活下来不够，得知道自己是"谁"。

面板里那张大陆地图还是灰的，可我看得见——赢了蛮子，是开局，远没到终局。四周没亮的地方，迟早有人占。咱们不立名，等于把地盘白白让出去。文明6里我吃过这亏：慢一步，邻国就把城池和奇观抢光了。

大壮蹲在石堰边磨斧，远远冲我扬了扬下巴。这人如今看我的眼神，跟头回见时两样——那会儿是打量，现在是认。老族长拄着棍在火塘边转，嘴里念叨着什么，像在数落，又像在谢。

有个事，我越想越硌得慌。

这帮人，跟我同生共死了一场，连个名号都没有。面板里写着「文明：华夏」，可他们自己，还只是「部落」「咱们」「族人」。一盘棋，连棋子叫什么都不知道，怎么下？文明6里开局第一桩，就是给自己的文明起名——我当初随手填的「华夏」，如今得让这片土，配得上这两个字。

我把众人聚到火塘边。

"咱们的名，该立了。"我开口，嗓子还哑着。

底下嗡一声。有人挠头，有人看老族长。阿陶眼睛亮得不行，凑过来："首领，叫啥？"

"华夏。"我说。

大壮皱眉："名字能当饭吃？"

角落里老石头嘟囔："叫啥不都挨饿。"

"当不了饭，"我扫过他们，"可当旗。旗立起来，人才知道往哪走。以前咱们是'那支部落'，挨打没人记得；往后，提起华夏，邻近的得掂量。你们记不记得，蛮子压境那夜，咱们连个叫得响的号子都没有？四十七个人，散沙一盘。从今起，这盘沙，有个名兜着了。"

老族长慢慢点头，棍子顿了顿："华夏……老辈人口里，华是繁盛，夏是广大。这名字，配得上。"

我心里一动。华是繁盛，夏是广大——这俩字，搁文明6里只是个标签；搁这片土上，是我要带着四十五个人活成的样子。标签，得用血去填。

蓝光落进阿陶眼睛里，这孩子喉头动了动，难得地安静了。大壮盯着那俩字，半晌憋出一句："写得齐整。"——他那人，夸人就这么费劲。

我借面板把「华夏」两个字调出来，蓝光悬在火塘上头。族人仰头看，那光落进每个人眼里，像把散沙照成了堆。那蓝光晃着，照得每张脸都亮了一层。我忽然想起文明6里点亮科技时的金光——可这回，光是从人堆里自己冒出来的。

"从今往后，"我声音不高，可谷里静得能听见风，"咱们不叫'那支部落'。咱们是华夏。蛮子再来，华夏接着砍；邻近部落，华夏接着收。这世上，得有咱们的名号。"

话说完，没人喊，可一片拳头攥了起来。老石头闷了半晌，忽然把石锄往地上一戳："成。华夏就华夏。"这老头，头回没跟我拧着。大壮把斧往地上一顿，第一个闷声："华夏。"他咧了咧嘴，那张凶脸在火光里松了点，"这名，够硬。"

大壮这人，认死理。他认一个名，就会替这名为它拼命。我见过他在谷口守三夜不闭眼，如今这股劲，有了去处。火塘边的气氛也变了——从前他们聚在一起，是等挨打；如今聚在一起，是等下一道令。这差别，比多十张弓顶用。等令的人，比等死的人难对付——也更可靠。

阿陶跟着喊，破音，这孩子喊得嗓子劈了也不停，眼眶红红的。老族长闭了眼，嘴唇抖。一个接一个，华夏、华夏、华夏——声浪撞在山谷石壁上，来回滚。我看着那圈蓝光，忽然觉得，这棋子，算是落定了名。文明6里，起名只是开场动画；在这儿，起名是宣战书。从今往后，盖亚大陆的棋盘上，多了一个叫华夏的玩家。轮到我们落子了。

我回头望了眼坡后的土堆。那两个没回来的，听不见这声"华夏"了。可他们的名，得有人替他们立着。这旗，也有他们一份。

名立了，日子得换个活法。我晚间把各处活计重新分了工——猎手归大壮，窑事归阿陶，粮仓归老族长的人。四十五个人，头回有了同一个姓，活也归到了一处。

夜里，我独自翻看面板。【文明：华夏】那行字，头回有了分量。声望+5，旁边又浮起一行：邻近部落态度——观望。观望就好。观望说明他们听见了，还没敢动。我点开文明详情，人口、粮食、科技、军力，一列数据静静躺着。从前看这些是任务栏，如今看，是四十五条命的账本。华夏这名，得对得起这串数。我关了面板，火塘的光映在腕上那圈印里。它静着，可我知道，从今夜起，它记的是华夏，不再是无名部落。

我望向神树林的方向。陶片上的弯钩在脑子里转。那处窑，蛮子去过，前任首领也牵扯其中。华夏立了名，该去把那处谜底掀了。我把念头跟大壮一说，他只回一句："去。名立了，就得让人知道华夏的脚踩在哪儿。"他顿了顿，"带上我。"我点头。神树林那处，宜快不宜拖。我连夜在面板上标了路线，粮、水、火把，一项项列清。华夏的出征，得从这一趟起，像个样子。

天边那道骑马的身影，还在不在？我调出面板，那枚「未识别文明」的标记仍亮着——坐得稳，马步匀，是常走远路的人。华夏刚立名，第一个邻居，就要上门了。是友是敌，不看脸面，看华夏长得够不够快。文明6里扩张靠的是快——这道理，落到地上一样。我合上眼，谷里的声浪还在耳朵里滚。华夏，这名，算立住了。

下一子，落在神树林。棋局这张盘，华夏算是正式落了座。
