## 四、核心设定

- **世界观**：超级大陆「盖亚」，受「棋盘规则」约束，文明发展被量化为数据，融合文明V/VI核心机制
- **主角**：常阳，26岁互联网产品经理，文明系列5000+小时铁粉，加班猝死穿越
- **金手指**：「文明面板」——可视化科技树/政策树/资源/军力等全部游戏化数据
- **两界联系**：每建成一座奇观，与现实世界的通道松动一分——这是回家的线索

## 五、核心主线（卷一）

```
穿越觉醒 → 种田起步 → 蛮族威胁 → 第一次战争（成长转折）
   → 技术革命（制陶/驯马/铁器）→ 外交博弈（白狼/炎部落）
   → 长城建成（第一个奇观）→ 古典时代 → 围城之战（卷一高潮）
   → 世界议会（新纪元开启）
```

## 六、群像体系

避免主角独狼，建立「华夏智囊团」：
- **常阳**：领袖（穿越者/面板持有者）
- **大壮**：军事统帅
- **阿陶**：首席工程师（科技线）
- **巴图**：外交官
- **（后续）史官**：文化线
- **（后续）谋士**：内政线
- **（后续）女性角色**：后宫线（多文明）

## 七、双线并行结构

- **明线（文明世界）**：常阳领导华夏文明从部落走向星际
- **暗线（现实世界）**：每卷1-2章，常阳家人/同事发现他失踪，寻找线索；建成奇观时现实出现异象

## 八、核心爽点模式

1. **面板数据暴涨**（种田建设快感——前期主导）
2. **以少胜多**（游戏思维战术）
3. **文明碾压**（装备制造升级）
4. **奇观建成**（大陆公告仪式感）
5. **外交格局**（从被围堵到四方来朝）
6. **战争碾压**（中后期加重）
7. **星际探索**（后期主导）

## 九、核心悬念（长期伏笔）

- 面板的真正来源
- 穿越的真相（与最后一局游戏的关系）
- 棋盘规则从何而来
- 上一纪元文明为何覆灭
- 是否有其他穿越者
- 两界通道的真相
- 宇宙级文明博弈（卷四后展开）

## 十、结局方向

主角带领华夏文明成为星际文明，找到两界通道真相，在「留下」与「回家」之间做出最终选择。

（具体结局在卷七-八创作中逐步明确）
