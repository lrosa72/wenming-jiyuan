## 四、章节排序断裂修复（重点）

**现象**：原 `正文存稿/` 中 Ch117「陈述」与 Ch118「投票」写的是**议会进行中**（印记发光、投票通过），而 Ch119「议会开幕前夜」、Ch120「记名台上的光」写的是**议会前夜**（阳城最后一晚、天亮议会开幕），Ch121 标「卷一终章」且"投票结果已宣布"。按原文件序阅读，读者会先看到"投票通过"再看到"议会前夜"，时序倒挂。

**根因**：上一轮"1–123 顺序重命名"采用的「主文件→b→c」排序逻辑，把本应更早的"前夜"场景（原 b/c 稿）排到了"议会进行中"场景之后，造成尾部错序。

**修复**：交换 117↔119、118↔120，使阅读序为：
```
114 第八个 → 115 七道光柱 → 116 石头圈
→ 117 议会开幕前夜（阳城最后一晚）
→ 118 记名台上的光（天亮·议会开幕）
→ 119 陈述（进入议会大厅·开口）
→ 120 投票（华夏通过）
→ 121 卷一终章·世界议会（结果宣布）
→ 122 离场 → 123 回家
```
文件名与 H1 标题同步更新，git 可回滚。
